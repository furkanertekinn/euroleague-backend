# Euroleague Information-Backend

## Özet:
### Node.js ve PostgreSQL kullanılarak rest api servisler sunan bir backend projesedir.

## Kullanılan Kütüphaneler:
- cors - 2.8.5
- express - 4.18.2
- pg - 8.11.3
